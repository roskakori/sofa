History

(First entry refers to current version, older versions see below.)

Version 1.1, 10-Apr-2000

  • Added C compiler support for Dice and Vbcc; both are available from Aminet.
  • Added sofa library for some basic AmigaOS stuff: inter process communication (ARexx), command line parsing (ReadArgs), pattern matching (MatchPattern)...
  • Updated to SmallEiffel -0.76b1.
  • Changed archive so that it does not anymore contain sub-archives that are extracted during installation.
  • Improved installation: checks system and environment for requirements, and reports about problems; uses SmallEiffel's internal "install.e" to setup the compiler, class loadpath etc; some messages and help texts should be more clear now.
  • Removed compile_to_jvm because there is no useful JVM for AmigaOS anyway.

Version 1.0, 12-Oct-1999

  • First public release
